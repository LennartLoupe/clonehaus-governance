## [0.1.0] - 2026-03-13

### Added
- Initial plugin release
- Interactive charter creation flow
- 7 framework exports (LangChain, N8N, CrewAI, AutoGen, LangGraph, Semantic Kernel Python/C#)
- EU AI Act compliance checker
- Ratification flow with accountability records
- Governance skill that auto-fires during agent setup
- Complete documentation and examples

### Features
- `/create-charter` - Guided charter creation
- `/export-context` - Framework code generation
- `/check-compliance` - EU AI Act validation
- `/ratify` - Accountability transfer

### Frameworks Supported
- LangChain (Python)
- LangGraph (Python)
- CrewAI (Python)
- N8N (JSON workflow)
- AutoGen (Python)
- Semantic Kernel (Python & C#)
